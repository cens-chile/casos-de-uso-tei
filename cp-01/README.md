## CP-01.Paciente atendido en nivel secundario

Este caso de prueba valida que el proceso de atención médica en el nivel secundario se ejecute de manera adecuada. Se verifica que el médico especialista acceda a la solicitud de interconsulta (SIC), registre correctamente el diagnóstico, las indicaciones y el tratamiento, y comunique esta información de forma oportuna al equipo de atención primaria.
